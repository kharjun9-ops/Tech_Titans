Progress Update (Hackathon Build)

2) Accuracy Improvements
- Improved weapon detection reliability using "weapon confirmation" (requires the weapon to persist across multiple frames before triggering an alert).

3) Performance / Demo Smoothness

4) Demo UI Polish
- Added a clean HUD panel with FPS + status + key controls (h=HUD, b=boxes, r=zone, q=quit).
- Added a clear "ALERT TRIGGERED" banner when an alert fires.
